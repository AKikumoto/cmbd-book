PsyToolkitで簡単な２腕バンディット課題を作る

(現在作成中) 手軽にオンライン実験・調査環境を構築できるPsyToolkitを使って，簡単な2腕バンディット課題を作る方法を記します。より実用的な課題の作り方はこの拡張として改めて記す予定です。

PsyToolkitにログインする

  • https://www.psytoolkit.org/cgi-bin/psy2.5.1/ にアクセス
  • 最初はRegister a PsyToolkit accountからアカウントを作る必要があります。

experimentを作る

  • exp_2banditというexperimentを作ります。

  • 以下をコードのボックスにペーストします。

options
  mouse on

fonts
  arial 36

task 2armed_bandit
  # 1000ms (1秒) 待つ
  delay 1000
  #
  # 選択肢の呈示 -------------------------------------------
  #
  # x = -150, y = 0の位置に，幅50，高さ50の緑の正方形を呈示
  show rectangle -150 0 50 50 0 255 0
  # x = -150, y = 0の位置に，幅50，高さ50の青の正方形を呈示
  show rectangle 150 0 50 50 0 0 255
  #
  # マウスでの選択の読み取り -------------------------------
  #
  readmouse l 1 99999 range 1 2
  set $a bitmap-under-mouse MOUSE_X MOUSE_Y
  #
  # 反応時間の記録
  set $decisiontime RT
  #
  # 選択の結果の決定，呈示 ---------------------------------
  #
  # 報酬の有無の決定のため，1から100の間の乱数を生成
  set $x random 0 100
  if $a == 1 # 選択肢1 (緑) を選択
    show text "▲" -150 60
    delay 500
    if $x <= 70 # 0.7の確率で報酬有り
      show text "+10ポイント" 0 -100
      set $reward 1
    fi
    if $x > 70  # 0.3の確率で報酬無し
      show text "+0ポイント" 0 -100
      set $reward 0
    fi
  fi
  if $a == 2 # 選択肢2 (青) を選択
    show text "▲" 150 60
    delay 500
    if $x <= 30 # 0.3の確率で報酬有り
      show text "+10ポイント" 0 -100
      set $reward 1
    fi
    if $x > 30 # 0.7の確率で報酬無し
      show text "+0ポイント" 0 -100
      set $reward 0
    fi
  fi
  delay 1000
  #
  # 1試行分のデータの保存-----------------------------------
  # 1列目: 反応時間 (選択肢が出てからキーを押すまで)
  # 2列目: 選択 (1 or 2)
  # 3列目: 報酬の有無 (0 or 1)
  save $decisiontime $a $reward

block main
  tasklist
    # タスク2armed_banditを20試行行う
    2armed_bandit 20 fixed
  end
  • saveをクリックします。
  • Compileをクリックします。

上で作ったexperimentを含めたsurveyを作る

  • minimum_2banditというsurveyを作ります。

  • 以下をコードのボックスにペーストします。

# 参加者の基本情報 ------------------------------------------

l: sex
t: radio
q: あなたの性別を選択してください。
- 男
- 女

# 2腕バンディット --------------------------------------------

l: 2armed_bandit
t: experiment
q: 下に出てくる赤い画面をクリックすると課題が始まります。マウスで2つの選択肢のうちいずれかをクリックし，できるだけ多くのポイントを稼いでください。
- exp_2bandit
  • Survey statusで，Should this survey be online accessible (check, even if you just want to test it)? にチェック
  • Study is being designedを選択します。
  • Change survey statusをクリックします。
  • Link to survey (you can copy this and send to other people): に表示されるURLにアクセスすると，調査が開始されます。

データのダウンロード

  • 作成したsurveyのページの， Prepare and download participant dataで Prepare datafiles for downloadをクリック。
  • Download data in zip fileボタンができたらクリックするとzipファイルがダウンロードされます。

追加予定の拡張

  • 反応時間に制限を設ける，反応ミスには警告メッセージを出す
  • 選択肢に画像ファイルを使う，固視点も表示する
  • 選択肢の呈示位置をランダマイズする
  • 報酬の有無をテーブルとして，Rで出力したテキストファイルで指定する (上のプログラムは毎試行乱数を振って報酬の有無を決めていますが，あらかじめ決めた割合で報酬が出る系列をシャッフルした方が参加者間の報酬割合のばらつきが抑えられます)
